# ♦️ 6 宽和高

```CSS``` 描述一个元素的大小，需要用两个属性

```width``` 和 ```height``` 即：宽和高

### 栗子

```css
html {
  width: 100%;
  height: 100px
}

body {
  width: 100rem;
  height: 100em
}
```

```width``` 和 ```height``` 的值有 4 种类型 

```px```、 ```百分比```、 ```rem``` 和 ```em```

## 小知识点

```CSS``` 里横的那条是宽，竖的那条是高，记住咯
