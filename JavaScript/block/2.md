### 重点掌握

```=``` 赋值操作符

### 栗子

```
var a = 20
```

## 小知识点

```+``` ```-``` ```*``` ```/``` 无异于日常数学运算符

但是 ```=``` 在 JS 里是赋值，不是 ```等于```

JS 里的 ```等于``` 是一种 ```布尔运算``` 

如： ```==``` 、 ```===```
