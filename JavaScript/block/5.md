# ♦️ 5 实操

分别申明类型为 ```Number``` ```String``` ```Array``` ```Object``` 的 4 个变量

## 栗子

### Number

```
var num = 2017
```

### String

```
var str = "晓程序公开课"
```

### Array 

```
var arr = [num, str]
```

### Object

```
var obj = { x: num, y: str, z: arr }
```

## 小知识点

掌握了四大基本数据类型，便可以根据实际需求为数据选择一种最恰当的存储方式
