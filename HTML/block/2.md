# ♦️ 2 HTML 文档

```HTML``` 文档有一个固定的结构，包含 5 对基本标签，一个格式声明

```<!DOCTYPE>```

```<html />``` 

```<head />``` 

```<meta />``` 

```<title />``` 

```<body />```

### 栗子

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8" />
    <title>标题</title>
  </head>
  <body>
  </body>
</html>
```

## 小知识点

```HTML``` 的文档结构，是固定的。

譬如：每一个 ```Web``` 页面都至少包含 ```栗子``` 里的 5 对标签 和一个格式声明。
