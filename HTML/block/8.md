# ♦️ 8 h1 标签

```<h1 />``` 标签有一个系列，分别是 1，2，3，4，5，6

h 是 head 的缩写，表示头部，顾名思义 ```<h1 />``` 标签一般被用来表示标题

h 后面的数字 1，2，3，4，5，6 表示标题的权重，或者说字号的大小。

所以数字越小，字号越大，权重越高。

### 栗子

```html
<h1>我是最大的标题</h1>
<h2>我是老二</h2>
<h3>所以我只能是老三了</h3>
<h4>请叫我四爷</h4>
<h5>辣么我就是龙五了</h5>
<h6>倒数第一也是第一</h6>

```

## 小知识点

```<h1 />``` 系列的标签非常好掌握，学一个就懂 6 个

不过呢，h 系列可以说是 ```HTML``` 里有且仅有的了

所以后面的标签，还得扎扎实实一个一个积累
